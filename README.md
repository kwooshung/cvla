npm，add 命令在 npm v5.0.0 中引入，这是在 2017 年。在此之前，你需要使用 npm install 命令来添加依赖。
yarn，add 命令在初始版本（v0.15.0，2016 年）中就已经存在。
pnpm，add 命令在初始版本（v0.2.0，2016 年）中就已经存在。
由于 npm、yarn、pnpm 较新的版本中，支持 add 命令来安装依赖，所以暂时不做兼容。
